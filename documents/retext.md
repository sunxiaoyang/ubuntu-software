# 安装 ReText
>ReText 是专用于 MarkDown [1] 语法的轻量级文本编辑器，你可以通过它采用 MarkDown 语法来编辑文档，并可实时预览。同时支持导出到 PDF、OTD 及 HTML 文件。其它主要功能包括支持多标签、HTML 语法高亮、快速插入 Tag 等等。

Ubuntu下可以使用命令直接安装
>sudo apt-get install retext


>#####若源中没有的retext，可以先添加源
>sudo add-apt-repository ppa:mitya57   
>sudo apt-get update   
>sudo apt-get install retext
